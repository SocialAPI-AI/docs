title: Use with Claude

SocialAPI.AI exposes an [MCP](https://modelcontextprotocol.io) (Model Context Protocol) server. Once connected, Claude can list accounts, fetch comments, reply, send DMs, and more — all through natural language.

## Prerequisites

- A SocialAPI.AI account with at least one connected social account
- A Claude plan that supports connectors (Pro, Max, Team, or Enterprise)

## 1. Open Connectors

Click **Customize** in the left sidebar.

<img
  src="/images/mcp-claude-settings.png"
  alt="Claude left sidebar with the Customize item highlighted"
/>

<Note>
  If you don't see Connectors under Customize yet, go to **Settings** → **Connectors** — they are in the process of moving to the Customize page.
</Note>

## 2. Add a custom connector

Click on the `+` icon to add a new connector and select `Add custom connector`.

<img
  src="/images/mcp-claude-add-server.png"
  alt="Add custom connector dialog in Claude"
/>

On the modal, enter the server name and URL:

```
SocialAPI.ai
```

```
https://api.social-api.ai/mcp
```

Click **Add**.

<img
  src="/images/mcp-claude-add-server-modal.png"
  alt="Add custom connector dialog in Claude"
/>


## 3. Authorize with your account

Claude will initiate an OAuth flow. A browser window will open to the SocialAPI.AI authorization page.

<img
  src="/images/mcp-claude-oauth.png"
  alt="SocialAPI.AI OAuth authorization screen"
/>

Log in with your SocialAPI.AI credentials and click **Authorize**. The window closes automatically when done.

<Note>
  This uses OAuth 2.1 with PKCE. No API key is required — your session is handled automatically.
</Note>

## 4. Enable in a conversation

Open a new Claude conversation. Click the **+** icon at the bottom-left of the input, select **Connectors**, and ensure **SocialAPI.AI** is toggled on.

<img
  src="/images/mcp-claude-conversation.png"
  alt="Claude conversation using SocialAPI.AI tools to list accounts"
/>

Try asking:

> "List my connected social accounts"

> "Show me the latest comments on my Instagram posts"

> "Reply to comment sapi_cmt_xxx with 'Thanks for your feedback!'"

## Available tools

Claude has access to 31 tools covering all SocialAPI.AI functionality:

| Category | What Claude can do |
|---|---|
| **Accounts** | List connected accounts, view platform metadata |
| **Posts** | List posts, create, update, delete, retry |
| **Comments** | Fetch comments, reply, moderate, toggle comments on/off |
| **DMs** | List conversations, read threads, send messages |
| **Reviews** | Fetch reviews, reply |
| **Mentions** | Fetch mentions |
| **Usage** | Check credit usage and limits |
| **Keys** | List and create API keys |
| **Webhooks** | Manage webhook endpoints |

## Troubleshooting

**Authorization fails or loops** — Make sure pop-ups are not blocked in your browser when Claude opens the OAuth window.

**Connector not showing in conversation** — Click the **+** icon in the chat input and enable SocialAPI.AI under **Connectors**.

**Tool calls fail with 401** — Your session may have expired. Open **Customize** in the left sidebar, remove SocialAPI.AI, and re-add it.
