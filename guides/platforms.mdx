All platforms and capabilities are available on every plan, including the Free tier. Plans differ only by [resource limits](/guides/rate-limits) (accounts, posts, interactions, storage).

## Media library

All plans support media uploads for post publishing. Uploaded media is stored in your personal media library and can be reused across posts.

| Plan | Storage limit |
|---|---|
| Free | 100 MB |
| Starter+ | Unlimited |

Manage your media library via the `/v1/media` endpoints - list, delete, and check storage usage. See the [API Reference](/api-reference) for details.

## Unsupported operations

When you call an endpoint for a capability a platform does not support, [SocialAPI.AI](https://social-api.ai) returns `501`:

```json
{
  "error": "platform does not support this operation",
  "code": "not_supported"
}
```

Your code should handle `501` gracefully - it is not an error in your integration, it is a platform limitation.
