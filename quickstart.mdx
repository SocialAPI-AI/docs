## Prerequisites

- A SocAPI API key — get one at [socapi.ai](https://socapi.ai)
- At least one connected social account (see Step 1 below)
- `curl` or any HTTP client

## Step 1: Connect a social account

<Note>
  OAuth connect is not yet live. During the beta, accounts are connected manually.
  Contact support@socapi.ai to connect your first account. You'll receive an account ID like `acc_01HZ9X3Q4R5M6N7P8V2K0W1J`.
</Note>

## Step 2: List your connected accounts

```bash
curl https://api.socapi.ai/v1/accounts \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json
{
  "count": 1,
  "data": [
    {
      "id": "acc_01HZ9X3Q4R5M6N7P8V2K0W1J",
      "platform": "instagram",
      "name": "Acme Corp",
      "username": "acmecorp"
    }
  ]
}
```

## Step 3: Fetch recent comments

```bash
curl "https://api.socapi.ai/v1/accounts/acc_01HZ9X3Q4R5M6N7P8V2K0W1J/comments?limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json
{
  "count": 10,
  "data": [
    {
      "id": "socapi_cmt_aW5zdGFncmFtOjE3ODQx",
      "platform": "instagram",
      "type": "comment",
      "author": { "name": "Jane Smith", "id": "17841405793187218" },
      "content": { "text": "Love this product!" },
      "created_at": "2026-02-15T14:30:00Z"
    }
  ]
}
```

## Step 4: Reply to a comment

Use the `id` from the previous response. The prefix `socapi_cmt_` tells SocAPI this is a comment — no extra parameters needed.

```bash
curl -X POST \
  "https://api.socapi.ai/v1/accounts/acc_01HZ9X3Q4R5M6N7P8V2K0W1J/interactions/socapi_cmt_aW5zdGFncmFtOjE3ODQx/reply" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text": "Thanks Jane! We are glad you enjoyed it."}'
```

```json
{
  "id": "socapi_cmt_cmVwbHk6MTIzNDU2Nzg5",
  "created_at": "2026-02-15T14:35:00Z"
}
```
